# GG Academy — «Дело о вертикали mVAS»

Интерактивный онбординг-курс по вертикали **mVAS** (mobile value-added services) для CPA-платформы Golden Goose. Формат — детективный RPG: учащийся в роли методолога-детектива расследует «дело о вертикали», проходя шесть модулей от экосистемы до финального экзамена.

Курс разработан как методологический артефакт: демонстрирует проектирование обучающего опыта (LXD) на незнакомом предметном материале — от анализа аудитории и прогрессии знаний до выбора игровых механик под конкретные дидактические задачи.

---

## 🎯 Запуск

Открой **`index.html`** в браузере — это главный экран (хаб) со всеми шестью модулями, встроенными внутрь. Никаких зависимостей, сервера или сборки не требуется: всё работает локально из одного файла.

Прогресс сохраняется автоматически в `localStorage`.

### GitHub Pages

Репозиторий готов к публикации через GitHub Pages:

1. **Settings → Pages → Source:** ветка `main`, папка `/ (root)`
2. Курс будет доступен по адресу `https://<username>.github.io/<repo>/`

## 📂 Структура

```
gg-academy/
├── index.html                          ← хаб (все модули встроены, точка входа)
├── program.html                        ← карта и программа курса
├── modules/                            ← модули по отдельности
│   ├── module-1-ecosystem.html         М1 · Экосистема и роли
│   ├── module-2-business-model.html    М2 · Бизнес-модель, EPC, офферы
│   ├── module-3-tracking.html          М3 · Трекинг, постбек, трекеры
│   ├── module-4-approval.html          М4 · ГЕО, операторы, фрод
│   ├── module-5-communication.html     М5 · Кризисная коммуникация
│   └── module-6-final.html             М6 · Compliance, экзамен
└── README.md
```

`index.html` самодостаточен — модули в папке `modules/` дублируются для отдельного просмотра и редактирования.

## 🎨 Дизайн

- **Стиль:** «Сепия-досье» — единая светлая палитра тёплой бумаги, без перепадов яркости
- **Атмосфера:** детектив-нуар через зерно бумаги, виньетки, карточки-досье, печатные шрифты
- **Типографика:** Oswald (заголовки), Special Elite (дневник, заметки), Inter (текст), IBM Plex Mono (метки)
- **Акцент по модулям:** золото, бирюза, тил, амбер, пурпур

## 🛠 Технологии

Чистый HTML/CSS/JavaScript без фреймворков и сборки. Каждый модуль — самодостаточный файл. Хаб встраивает модули через `iframe srcdoc`, обмен прогрессом — через `postMessage`.

## 🎮 Механики
